📝 Project Introduction

This is a deep personality matching tool specifically designed for players of "Gakuen Idolmaster." By utilizing 40 behavioral psychology-based scenario questions, it constructs an 8D personality model that integrates MBTI cognitive functions (E/N/T/J) with Idol-specific traits (Confidence, Ambition, Empathy, and Uniqueness).

The system employs a Gaussian Similarity Decay algorithm to calculate the Euclidean distance between the user's personality coordinates and those of 14 characters from Hatsuboshi Gakuen (including idols, teachers, and the student council president), resulting in a high-resolution resonance report and interactive radar charts.
