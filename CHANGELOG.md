🚀 v1.2.0 Algorithm Upgrade & UX Optimization

This update focuses on completely refactoring the matching algorithm to ensure every idol has a fair chance of appearing, alongside various UI and stability enhancements.

🌟 Core Optimizations

Algorithm Refactoring: Implemented a new hybrid algorithm using "Cosine Similarity (80%) + Euclidean Distance (20%)". The system now evaluates the "shape" (wavelength) of your personality rather than absolute extreme values. All 14 idols now have an equal and fair chance to reach Top 1!

Score Inflation Mechanism: Added a dynamic score boost logic. Your Top 1 resonance rate will now stably land in the 86% ~ 95% high-score range, providing a much better and more encouraging testing experience.

Radar Chart Upgrade:

Replaced the underlying dimensions with a standardized 8D model: E, J, T, C, M, O, A, U.

Removed redundant English letters from the radar chart for a cleaner interface.

Refined dimension terminology for better accuracy (e.g., changed Discipline to Judging/Plan, Optimism to Composure/Open-mindedness).

🐛 Bug Fixes

Fixed webpage crash issues caused by importing external icon libraries. All icons have been completely replaced with lightweight, built-in native SVG components.

Retrieved the "missing" idols: Sena Juo, Neo Asari, Ume Hanami, and Rinami Himesaki. All 14 idols are now back in the roster.

Fixed the issue where avatar images failed to load and updated them with more stable direct image links.

⚖️ Stat Balancing

Completely removed the overly extreme character stat settings from earlier beta versions.

Accurately fine-tuned Temari Tsukimura's Confidence dimension (C=6), as well as the social dimensions for Hiro Shinosawa and Mirei Hataya. Character profiles are now much closer to their official lore and story settings.
